# 数学大冒险 — 功能手册

## 这是什么

数学大冒险是一款面向小学 5-6 年级学生的计算能力训练工具。它把枯燥的计算练习变成了一场冒险——每答对一题就能获得经验值、解锁成就、提升等级。坚持每天练习，你会看到自己肉眼可见的进步。

打开方式：在电脑、手机、iPad 的浏览器中访问即可使用，无需安装。

---

## 快速开始

1. 首次打开，输入一个昵称，选择年级（5 或 6 年级）
2. 进入首页后，点击任意主题开始练习
3. 选择难度（普通/困难/魔王）和题目数量（10/15/20），点击「开始练习」
4. 逐题作答，完成后查看结算报告

## 八大练习主题

每个主题针对一类常见的计算薄弱点。所有主题从一开始就全部开放。

| 主题 | 练什么 | 题型 |
|------|--------|------|
| ⚡ 口算速算 | 整数加减乘除的心算速度和准确度 | 输入数字 |
| 🎯 数感估算 | 比大小、估算结果、四舍五入 | 选择题 / 输入数字 |
| 📝 竖式笔算 | 进位退位的每一步是否正确 | 逐位填空 |
| 🔄 运算律 | 交换律、结合律、分配律的识别和应用 | 选择题 / 输入数字 |
| 🔢 小数计算 | 小数对齐、移位、乘除中的小数点处理 | 输入数字 |
| 🔗 括号变换 | 增减括号时正确变换正负号 | 选择题 |
| 📊 多步计算 | 多步混合运算的运算顺序和抄写准确性 | 输入数字 |
| ⚖️ 方程移项 | 解方程时移项变号的准确性 | 输入数字 |

## 难度分级

三个难度档位，选择适合自己的挑战：

| 难度 | 说明 | 举例（口算） |
|------|------|-------------|
| 😊 普通 | 基础练习，打好根基 | 两位数加减、九九乘法表 |
| 😤 困难 | 有一定挑战，需要动脑 | 三位数加减、两位乘一位、混合运算 |
| 👹 魔王 | 最强难度，计算高手专属 | 大数混合运算、限时挑战 |

## 竖式笔算（特殊交互）

竖式笔算是唯一使用「逐位填空」交互的主题。界面会显示标准竖式格式：

```
进位:  [ ][ ¹][ ¹][ ]
       [ 3][ 8][ 4][ 7]
    +  [ 2][ 9][ 5][ 6]
    ─────────────────────
       [ _][ _][ _][ _]  ← 从右向左逐格填入
```

操作方式：
1. 系统从个位开始，高亮当前要填写的格子
2. 输入一个数字（0-9），系统**立刻判断对错**
3. 答对：格子变绿，进位标记自动出现，下一格激活
4. 答错：格子红色抖动，重新输入。连续错 2 次后显示提示
5. 全部填完即完成该题

手机上会显示数字键盘方便点击输入。

## 游戏化系统

### 经验值 (XP)

每答对一题获得基础 XP，公式：

```
基础 XP = 10 + (难度 - 1) × 5

难度1 = 10 XP，难度5 = 30 XP，难度10 = 55 XP
```

额外加成：
- **连击加成**：连续答对 5 题 ×1.2，10 题 ×1.5，20 题 ×2.0
- **速度加成**：在时限 30% 以内答对 → +25%
- **完美场次**：一次练习全部答对 → 结算时总 XP +50%

### 等级系统

共 40 个等级，升级所需 XP = 100 × 等级²。

| 等级段 | 称号 | 标志 | 升级所需总 XP |
|--------|------|------|--------------|
| 1-5 | 算术学徒 | 🌱 | 100 - 2,500 |
| 6-10 | 计算新星 | ⭐ | 3,600 - 10,000 |
| 11-15 | 数字猎手 | 🏹 | 12,100 - 22,500 |
| 16-20 | 运算达人 | 🏅 | 25,600 - 40,000 |
| 21-25 | 公式勇士 | ⚔️ | 44,100 - 62,500 |
| 26-30 | 方程侠客 | 🛡️ | 67,600 - 90,000 |
| 31-35 | 计算大师 | 👑 | 96,100 - 122,500 |
| 36-40 | 数学王者 | 🏆 | 129,600 - 160,000 |

### 连击 (Combo)

每连续答对一题，连击数 +1。答错归零。

| 连击数 | XP 倍率 | 界面提示 |
|--------|---------|---------|
| 1-4 | ×1.0 | 无 |
| 5-9 | ×1.2 | 黄色连击标签 |
| 10-19 | ×1.5 | 橙色连击标签 |
| 20+ | ×2.0 | 紫色连击标签 |

### 红心

每次练习有 5 颗红心。答错一题扣 1 颗。红心用完则练习提前结束。
练习之间红心自动回满，不需要等待。

### 连续学习 (Streak)

每天完成练习即可保持「连续学习天数」。

- 今天练了 → 连续天数 +1
- 昨天练了、今天还没练 → 等你今天练了就 +1
- 超过一天没练 → 连续天数重置为 1（除非有冻结令牌）

**冻结令牌**：每连续学习 7 天获得 1 个冻结令牌（最多存 2 个）。如果某天忘记练习，系统自动消耗一个令牌来保护你的连续记录。

### 每日目标

默认每天 50 XP。可在「个人中心」调整为 30 / 50 / 100 / 200 XP。首页顶部的圆形进度环显示今天的完成进度。

## 错题本

所有答错的题目会自动收集到「错题本」中。

- 按主题分组展示
- 显示你当时的错误答案和正确答案
- 显示解题思路和要点
- 最多保留最近 100 道错题

## 学习进度

进度页面展示三个维度的数据：

1. **本周学习**：近 7 天每天获得的 XP 柱状图
2. **学习日历**：近 30 天的学习热力图（绿色越深 = 当天练得越多）
3. **各主题进度**：每个主题的准确率、当前难度、已答题数、掌握度星级

掌握度星级（0-5 星）根据准确率和当前难度综合判定：
- ★☆☆☆☆ 已开始练习
- ★★☆☆☆ 准确率 ≥ 70%，难度 ≥ 3
- ★★★☆☆ 准确率 ≥ 85%，难度 ≥ 5
- ★★★★☆ 准确率 ≥ 90%，难度 ≥ 7
- ★★★★★ 准确率 ≥ 95%，难度 ≥ 9

## 数据存储

当前版本的练习数据保存在浏览器本地 (localStorage)：

- 换浏览器或清除浏览器数据会丢失记录
- 后续版本将支持 Supabase 云端同步，届时可跨设备使用同一账号
