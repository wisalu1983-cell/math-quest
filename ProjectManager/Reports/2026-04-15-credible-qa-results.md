## 三、发现并修复的 Bug

### Bug-1: 进阶结算页无限重渲染（P0 严重）

| 项 | 内容 |
|----|------|
| 位置 | `SessionSummary.tsx` → `AdvanceSummary` 组件 |
| 根因 | `useGameProgressStore(s => s.getAdvanceProgress(topicId))` 中 `getAdvanceProgress` 是方法调用，每次返回新对象，Zustand 浅比较判定为变化，触发无限 re-render |
| 影响 | 进阶 session 完成后白屏，React "Maximum update depth exceeded" |
| 修复 | 改为 `useGameProgressStore(s => s.gameProgress)` 然后在组件内派生星级数据 |
| 验证 | 修复后 T3-02~T3-10 全部 PASS |

### Bug-2: CR-002 竖式浮点精度（上一轮已修）

从 `a + b` 浮点运算改为从 `columns.length` 推算位数。
