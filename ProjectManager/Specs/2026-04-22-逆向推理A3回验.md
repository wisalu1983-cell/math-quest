## 一、回验目标

用"逆向推理第 1 关"检验 A3 总则能否正确指导题干设计：
- 若 A3 机制一的判断标准能说明为什么这道题有审题价值 → A3 站得住
- 若出现 A3 说不清楚的情况 → 标记为 A3 的缺口，需修订总则

## 二、第 1 关题干样本

第 1 关（`number-sense-S2-LC-L1`，difficulty=5）由 `generateReverseRound(d=5)` 生成，典型题干：

```
一个一位小数四舍五入到个位后是 7，这个数最大是多少？
```

```
一个一位小数四舍五入到个位后是 12，这个数最小是多少？
```

---

## 三、A3 机制一校验

**机制一的判断标准**（来自 A3 总则 §四）：  
> 把方法名从 prompt 里删掉后，题目仍然完整可解，且题干的语义情景是学生得出解法的唯一路径。

| 校验项 | 分析 | 结论 |
|---|---|---|
| **prompt 是否给出方法名？** | "四舍五入"出现在题干中，但它是**结果条件的陈述**（"四舍五入后是 7"），不是"请你用四舍五入法"的操作指令。去掉"四舍五入"这四个字后，条件本身不成立——学生需要知道这是四舍五入才能建立范围关系。 | ✅ 方法名是信息的一部分，不是操作指令；不违反机制一 |
| **B'（数量关系建立）是否是核心难点？** | 学生需要自行构造："`四舍五入到整数后是 7`" → "原数的千分位被取整" → "最大：取'舍'的最大值，7.4；最小：取'入'的最小值，6.5"。这个逆向推理链学生无法跳过，必须走 B' 才能答对。 | ✅ B' 是唯一解题路径，核心难点正确 |
| **不同理解会不会走向不同方法？** | 反向思考：如果学生理解成"进一法后是 7"，会推出原数在 [6, 7) 而非 [6.5, 7.4]，答案完全不同。说明理解语义条件的精确含义（确认是四舍五入而非其他取整方式）本身就构成 B 步骤的挑战。 | ✅ 通过 |

**结论：A3 总则站得住。** `reverse-round` 第 1 关的题干设计正确实现了机制一。

## 四、发现问题：hints 过度提示

**问题**：当前 hints 说：

```
最大到多少还能"四舍"（保持不进位）
最小到多少还能"五入"到这个数
```

这直接把 B' 的推理路径（"四舍取最大"/"五入取最小"）拆解出来告诉学生，相当于跳过了逆向数量关系的建立步骤。

**改进方向**：hints 应提供方向性引导，而不是直接给出规则分支。

| 当前 hint（过度提示）| 改进后 hint（方向引导）|
|---|---|
| "最大到多少还能'四舍'" | "四舍五入后恰好是 N——原数最大能走到哪里还不会进位到 N+1？" |
| "最小到多少还能'五入'到这个数" | "什么样的数，四舍五入后会'入'到 N？它最小是多少？" |

## 五、prompt 模板扩充（≥5 组）

当前生成器只有 1 套固定模板，容易被识别为"套路题"。扩充 5 组不同表述，覆盖相同的 B' 挑战。

### 模板 1（标准，当前已有）

```
一个一位小数四舍五入到个位后是 {N}，这个数{最大|最小}是多少？
```

### 模板 2（换用"近似值"概念）

```
某一位小数用四舍五入法保留到整数，近似值是 {N}，这个小数{最大|最小}是多少？
```

### 模板 3（直接问结果端点，省略"这个数"）

```
四舍五入到个位后等于 {N} 的一位小数，{最大|最小}可以是多少？
```

### 模板 4（填写具体数位，十分位填空形式）

```
{N}.□ 用四舍五入法取到个位后结果仍然是 {N}，□ 里最大能填几？
```

```
{N-1}.□ 用四舍五入法取到个位后，结果变成了 {N}，□ 里最小能填几？
```

> **说明**：模板 4 不写"这个数精确到个位"或"精确到个位"，因为"精确到个位"未指明方法（进一法/去尾法/四舍五入均可"精确到个位"），会引起歧义。模板 4 明确写出"四舍五入法"。

### 模板 5（故事化视角）

```
小明把一个一位小数用四舍五入法凑成整数后写下了 {N}，这个一位小数原来{最大|最小}可能是多少？
```

---

**5 组模板对比：**

| 模板 | 主语 | 核心句式 | 适合档位 |
|---|---|---|---|
| 1 | "一个一位小数" | "四舍五入到个位后是 N" | 低档（d=5） |
| 2 | "某一位小数" | "用四舍五入法保留到整数，近似值是 N" | 低档（d=5） |
| 3 | 省略主语 | "四舍五入到个位后等于 N 的一位小数" | 低档（d=5） |
| 4 | 填空形式 | "N.□ 用四舍五入取到个位后仍然是 N，□ 最大/最小填几" | 低档（d=5）|
| 5 | 故事化 | "小明把一位小数用四舍五入凑成整数写下了 N" | 低档（d=5） |

所有模板均：✅ 明确指定方法为"四舍五入"，✅ 不含操作指令，✅ 学生须通过 B' 建立范围关系。

## 六、高档扩展路径（基于 A3 引导）

A3 总则说明 `reverse-round` 的主训练步骤是 B'（逆向数量关系）。随档位升高，B' 的复杂度应增加：

| 关卡 | difficulty | B' 复杂度 | 新增挑战 |
|---|---|---|---|
| S2-LC-L1（第 1 关，本文验证对象）| 5 | 一位小数 → 整数范围 | B' 基础：[N-0.5, N+0.4] |
| S2-LC-L2（第 2 关）| 6 | 两位小数 → 一位小数范围 | 两层精度，跨百分位进位 |
| 高档扩展（可选）| 7+ | 三位小数 + 最大最小差值 | 同时求最大最小并计算两者之差（对应真题 `Q-A02-A-002`）|

**真题对齐验证（`Q-A02-A-002`）**：  
> "一些三位小数四舍五入后约是 0.80，则这些三位小数中，最大数与最小数相差（ ）"  
> 答：0.804 - 0.795 = 0.009

这类题需要学生：
1. 分别推导最大值（千分位最大取 4）和最小值（需跨位逆推：0.795）
2. 计算两者差

对应 A3 总则：B' 步骤需要两次逆向关系建立，难度显著高于第 1 关，适合高档。
