## Q-A04-F-001

**题目**: 用字母表示乘法交换律是（　　）。
- A. a + b = b + a
- B. a × b = b × a
- C. (a × b) × c = a × (b × c)
- D. (a + b) × c = a × c + b × c

**题型**: multiple-choice
**选项**:
- A. a + b = b + a
- B. a × b = b × a ✓
- C. (a × b) × c = a × (b × c)
- D. (a + b) × c = a × c + b × c

**答案**: B
**解题步骤**:
1. 乘法交换律是指两个数相乘，交换因数的位置，积不变
2. 用字母表示为 a × b = b × a

**知识点**: 乘法交换律的字母表达
**易错点**: 混淆交换律和结合律的字母表达式
**来源**: U2-小数乘除法.md
