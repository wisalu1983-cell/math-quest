## Q-A03-F-001

**题目**: 列竖式计算: 2.57 + 0.43

**题型**: vertical-fill
**答案**: 3
**解题步骤**:
1. 小数点对齐，个位: 7 + 3 = 10，写 0 进 1
2. 十分位: 5 + 4 + 1 = 10，写 0 进 1
3. 个位: 2 + 0 + 1 = 3
4. 结果为 3.00 = 3

**知识点**: 小数加法竖式，小数点对齐，连续进位
**易错点**: 忘记进位；小数点未对齐
**来源**: U1-复习与提高.md

**生成器参照**:
- 映射 TopicId: `vertical-calc`
- 映射子函数: `generateAdditionSteps()`
- 参数模式: 两位小数 + 两位小数，含连续进位
- 与生成器差异: 无

**标签**: `#小数加法` `#进位` `#小数点对齐`

## Q-A03-F-002

**题目**: 列竖式计算: 37.7 − 2.43

**题型**: vertical-fill
**答案**: 35.27
**解题步骤**:
1. 小数点对齐，37.70 − 2.43
2. 百分位: 0 − 3，不够减，向十分位借 1，10 − 3 = 7
3. 十分位: 7 − 1 − 4 = 2
4. 个位: 7 − 2 = 5
5. 十位: 3 − 0 = 3
6. 结果为 35.27

**知识点**: 小数减法竖式，位数不同需补零，退位减法
**易错点**: 被减数末尾未补零导致出错；退位计算错误
**来源**: U1-复习与提高.md

**生成器参照**:
- 映射 TopicId: `vertical-calc`
- 映射子函数: `generateSubtractionSteps()`
- 参数模式: 一位小数 − 两位小数，含退位和补零
- 与生成器差异: 无

**标签**: `#小数减法` `#退位` `#补零`
