# 真题库录入规范

## 单题格式

每道题目按以下 Markdown 格式录入：

```markdown
## Q-{主题编号}-{F|A}-{三位序号}

**题目**: {完整题目文本}

**题型**: {题型}
**答案**: {答案}
**解题步骤**:
1. {步骤1}
2. {步骤2}

**知识点**: {考查的知识点}
**易错点**: {学生常犯错误}（可选）
**来源**: {出处}
**来源页码**: {页码或年份}（可选）

**生成器参照**:
- 映射 TopicId: `{topicId}`
- 映射子函数: `{函数名}()`
- 参数模式: {描述}
- 与生成器差异: {差异说明或"无"}

**标签**: `#标签1` `#标签2`

---
```

## 字段说明

### 题号规则

格式: `Q-{主题编号}-{难度}-{序号}`
- 主题编号: A01 ~ D01
- 难度: F = foundation（打基础），A = advanced（进阶）
- 序号: 001 起，三位数字

示例: `Q-A07-F-001`（A07 多步混合运算，打基础，第 1 题）

### 题型（对齐 QuestionType）

| 值 | 说明 | 适用场景 |
|----|------|---------|
| `numeric-input` | 数字输入 | 计算题、填空题 |
| `multiple-choice` | 选择题 | 四选一 |
| `true-false` | 判断题 | 对错判断 |
| `vertical-fill` | 竖式填空 | 竖式笔算 |
| `multi-step-input` | 多步输入 | 分步计算 |
| `expression-select` | 表达式选择 | 运算律等价变换 |
| `show-steps` | 展示步骤 | 应用题（需列式） |
| `fill-blank` | 文字填空 | 概念类填空 |

### 必填字段

| 字段 | 必填 | 说明 |
|------|------|------|
| 题号 | 是 | 唯一标识 |
| 题目 | 是 | 完整题目文本 |
| 题型 | 是 | 对齐 QuestionType |
| 答案 | 是 | 最终答案 |
| 解题步骤 | 是 | 至少写出关键步骤 |
| 知识点 | 是 | 本题考查的核心知识点 |
| 来源 | 是 | 出处标注 |
| 生成器参照 | 是 | **核心字段**，与引擎的关联 |

### 可选字段

| 字段 | 说明 |
|------|------|
| 易错点 | 学生常犯错误，对陷阱设计有价值 |
| 来源页码 | 具体页码或试卷年份 |
| 标签 | 辅助检索的知识点标签 |

## 难度分级标准

### 打基础（foundation）

- 对应难度值 1-5
- 教材课本例题和课后练习水平
- 数值范围与教材一致
- 考查单一知识点
- 学生应当全部掌握的核心内容

### 进阶（advanced）

- 对应难度值 6-10
- 期末/毕业考试中等偏难题
- 小升初衔接题水平
- 至少包含一个进阶要素：
  - 多步骤综合运用
  - 非常规数值（大数、特殊小数）
  - 陷阱/易错设计
  - 多知识点交叉

## 特殊题型格式

### 选择题

```markdown
**题型**: multiple-choice
**选项**:
- A. 134 - 75 - 25 ✓
- B. 134 - 75 + 25
- C. 134 + 75 - 25
- D. 134 + 75 + 25

**答案**: A
```

### 几何题（含图形）

用文字精确描述图形，必要时用 ASCII art：

```markdown
**题目**: 一个直角三角形，两条直角边分别为 3cm 和 4cm，求面积。

   |\
 3 | \
   |__\
    4
```

### 应用题

题目文本控制在 3 行以内。

## 生成器参照字段填写指南

### 已有 TopicId（A01-A08）

```markdown
**生成器参照**:
- 映射 TopicId: `multi-step`
- 映射子函数: `generateReduceSubtract()`
- 参数模式: targets=[100], b+c=T, a>T
- 与生成器差异: 无，生成器已覆盖
```

### 新增主题（A09, B01-D01）

```markdown
**生成器参照**:
- 映射 TopicId: `[待建] fraction-ops`
- 建议题型: `numeric-input`
- 生成器设计要点: 参数化（分母范围、是否异分母），确保约分后为最简分数
```

## 质量检查清单

录入每道题后自查：

- [ ] 题号格式正确（Q-A07-F-001）
- [ ] 答案已手动验算
- [ ] 解题步骤完整清晰
- [ ] 来源已标注
- [ ] 生成器参照字段已填写
- [ ] foundation 题目确实是基础水平
- [ ] advanced 题目确实含进阶要素
- [ ] 无与同文件内已有题目重复
