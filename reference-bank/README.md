## 使用方式

1. **校准生成器**: 对比真题的数值范围、题型分布与现有生成器输出
2. **发现缺口**: 通过"生成器参照 > 与生成器差异"字段找到未覆盖的题型
3. **提取高频考点**: 统计真题中反复出现的数字组合（如 25×4, 125×8）
4. **设计陷阱**: 借鉴真题中的干扰项设计和"易错点"标注
5. **规划新主题**: A09-D01 六个新主题的生成器需求直接来自真题积累
