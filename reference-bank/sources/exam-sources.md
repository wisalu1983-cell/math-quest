## 一、免费可浏览（目录/标题/部分内容）

### 1. 21世纪教育网 (21cnjy.com)
- **内容**: 沪教版全册课件、教案、试卷、学案
- **优势**: 资源最全，按教材单元精确分类
- **限制**: 下载需积分/VIP，但目录和标题免费浏览
- **移动版**: mip.21cnjy.com（反爬较弱，便于抓取目录）
- **详细索引**: 见 [21cnjy-index.md](./21cnjy-index.md)

### 2. 教习网 (51jiaoxi.com)
- **内容**: 沪教版五四制课程目录结构清晰
- **优势**: 课程目录可直接浏览，按单元组织
- **限制**: 试卷内容需下载 PDF
- 五年级上册: https://www.51jiaoxi.com/books/3000662/
- 五年级下册: https://www.51jiaoxi.com/books/3000664/
- 期末押题卷示例: https://www.51jiaoxi.com/doc-16537963.html

### 3. 小学试卷网 (szxuexiao.com)
- **内容**: 免费小学试卷，大部分含答案
- **优势**: 声称免费下载，有沪教版五年级期末试卷
- **限制**: 页面内容编码有问题，部分文字显示异常
- 2022沪教版五下期末: https://www.szxuexiao.com/Examination/html/6222.html

### 4. 我的数学网 (mymaths.cn)
- **内容**: 1655+ 条五年级数学资源（试卷+练习）
- **优势**: 资源量大，有答案标注
- **限制**: 实际题目在 PDF 附件中，非 HTML 内嵌
- 五年级频道: https://www.mymaths.cn/6/

### 5. CSDN 博客
- **内容**: 教师/家长分享的练习题整理
- **优势**: 内容直接嵌入网页，可抓取

## 推荐的数据收集策略

1. **教材原题（P0）**: 直接从课本拍照/OCR，用 Mathpix Snip 识别
2. **考试真题（P1）**: 从**通通好课网**（免费）或 21cnjy.com 下载试卷转录
3. **练习补充（P2）**: 从知乎文章、CSDN 博客抓取已整理的题目集
4. **拔高题（P3）**: 从**上海奥数网**（免费）获取小升初真题
5. **批量采集**: 优先使用通通好课网和第一试卷网（均免费无需注册）
6. **上海真题**: 学科网上海专辑（付费但最全）或 Scribd 历年考卷
